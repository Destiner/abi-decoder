# abi-decoder
Decode Ethereum transactions online

Link: https://abi-decoder.netlify.com/

## Features

* Client-side
* Instant conversion for 500+ functions
* Metadata included
* 4byte fallback: 150,000+ signatures

## Local registry

* 500+ signatures from more than 15 contracts (DEXs, lendings, games, marketplaces, utility)
* Metadata from the smart contract source (parameter names)
* Tuple support (example: 0x v2 methods)

## Run locally

`npm run serve`
